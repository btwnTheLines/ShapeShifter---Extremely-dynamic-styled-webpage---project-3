How ShapeShifter works:

The styling of the page is changed using Javascript and CSS. The HTML is the basic skeleton which has the structure of the web page, so when you click on one of the theme buttons the HTML stays the same (but can be changed if the code is tweaked, but that is not the focus for ShapeShifter), however the CSS properties and/or CSS classes of the elements on the page are changed using Javascript to enable 'shape shifting' functionality.

CSS properties are specific styling properties of an element whereas a CSS class contains a number of CSS properties bundled together.
